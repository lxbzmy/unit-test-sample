# 单元测试示例

## 项目目的

本项目旨在学习单元测试（Unit Test），提供不同编程语言下的单元测试案例和示例。

## 目录结构

为简化运行，项目按 `语言/案例` 的结构组织，每个案例均可以独自运行

详见各语言目录的自述。

一些共同的约定

* 构建输出会统一使用 `taget` 目录
* 源码放 `src`
* 测试代码放 `test`
* 缩进使用2个空格
* 运行环境以ubuntu24的自带的apt源版本为基线

## 使用说明

每个案例文件夹包含相应的代码和测试文件。请根据需要查看和运行示例。

为方便演示，本项目可以将所有依赖项打包成container image 运行。
